## 프로젝트 소개

#### LUCKY PYNK
- 물건을 분실한 경우 글을 등록하여 분실물을 찾을 수 있는 플랫폼
- 물건을 습득한 경우 글을 등록하여 물건의 주인을 찾을 수 있는 플랫폼
- 회원가입 필수 (회원만 이용 가능)
- 키워드, 지역 별로 게시글 검색 가능
- 게시글에 댓글 기능 추가
- 쪽지 기능을 넣어 회원들간의 소통을 중시
